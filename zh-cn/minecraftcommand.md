##### 语法：
```

/reg <密码> <重复密码>
```

## /mail
##### 作用：管理玩家在服务器内的邮件。
##### 语法：
```
/mail read [页码] - 阅读你的邮件的第一（或指定）页
/mail clear [编号] - 清空所有（或指定）的邮件
/mail clear <玩家> [编号] - 清除给定玩家的所有或指定邮件
/mail clearall - 清除所有玩家的邮件
/mail send <玩家> <消息> - 向指定玩家发送给定消息
/mail sendall <消息> - 向所有玩家发送给定的消息
/mail sendtemp <玩家> <过期时间> <消息> - 向指定的玩家发送给定过期时间的临时消息
/mail sendtempall <过期时间> <消息> - 向所有的玩家发送给定过期时间的临时消息
```

<br>

## /res
##### 作用：计算手中（或指定物品）的价值。
##### 语法：
```
/res create <领地名称> - 创建一块领地，范围为当前选区。
/res remove <领地名称> - 删除一块领地（仅限领地所有者）。
/res list - 查看你拥有的所有领地。
/res info - 查看当前领地的详细信息。
/res set <权限> <true/false> - 设置领地的权限（如build、destroy等）。
/res set - 直接输入/res set来查看所有可设置的权限（有箱子菜单）。
/res pset <玩家名> <权限> <true/false> - 为特定玩家设置领地权限。
/res tpset - 设置领地的传送点。
/res tp <领地名称> - 传送到指定领地。
/res subzone <子领地名称> - 在当前领地内创建一个子领地。
/res expand <方向> <格数> - 向指定方向扩展领地范围。
/res contract <方向> <格数> - 向指定方向缩小领地范围。
/res give <玩家名> <领地名称> - 将领地转让给其他玩家。
/res rename <新名称> - 重命名你的领地。
/res message <enter/leave> <消息> - 设置玩家进入或离开领地时的提示消息。
/res message clear - 清除领地进入/离开的提示消息。
/res trust <玩家名> - 允许某玩家在你的领地内建造和破坏。
/res untrust <玩家名> - 撤销某玩家在你的领地内的建造和破坏权限。
/res ltrust <玩家名> - 允许某玩家在你的领地内使用容器（如箱子）。
/res unltrust <玩家名> - 撤销某玩家在你的领地内使用容器的权限。
/res atrust <玩家名> - 允许某玩家在你的领地内使用所有权限。
/res unatrust <玩家名> - 撤销某玩家在你的领地内的所有权限。
/res flag <flag> <true/false> - 设置领地的全局标志（如pvp、explosion等）。
/res limits - 查看你的领地数量和使用情况。
/res version - 查看领地插件的版本信息。
```
